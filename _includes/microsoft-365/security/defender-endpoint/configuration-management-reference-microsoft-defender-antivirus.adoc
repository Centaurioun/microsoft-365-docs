Microsoft Defender Antivirus]

cmdlets]

Instrumentation (WMI)]

link:./command-line-arguments-microsoft-defender-antivirus.md[Microsoft
Malware Protection Command Line Utility] (referred to as the
_mpcmdrun.exe_ utility
